You are given a list of files. You need to sort this list by the file extension. The files with the same extestion should be sorted by name.

Some possible cases:

  • Filename cannot be an empty string;
  • Files without the extension should go before the files with one;
  • Filename ".config" has an empty extenstion and a name ".config";
  • Filename "config." has an empty extenstion and a name "config.";
  • Filename "table.imp.xls" has an extesntion "xls" and a name "table.imp";
  • Filename ".imp.xls" has an extension "xls" and a name ".imp".

Input: A list of filenames.

Output: A list of filenames.
